# AI-Graded

## Repository name: AI-Graded
## Project name: Best Cars Dealership - Full Stack Django/React Application

This is a Full Stack Application Development Capstone Project built with Django and React.

### Features
- Django REST Framework backend
- React frontend
- Car dealership management
- Dealer reviews with sentiment analysis
- User authentication (login/logout/register)
- Admin panel

### Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
